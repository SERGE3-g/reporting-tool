1. Introduzione

Il Tool di Reporting Commissioni Mensile è un'applicazione desktop progettata per semplificare l'estrazione e l'analisi dei dati di commissione contenuti in file JSON, XML e CSV. Questo strumento consente di estrarre automaticamente informazioni chiave, applicare filtri, visualizzare statistiche e esportare i risultati in diversi formati.

Interfaccia principale dell'applicazione

Con questa applicazione potrai:

  • Importare facilmente file JSON, XML e CSV tramite drag & drop
  • Estrarre automaticamente campi come descrizioni, importi, IBAN e altri dettagli
  • Filtrare i dati in base a diversi criteri
  • Visualizzare i risultati in una tabella ordinata
  • Esportare i dati in formato CSV, XLSX o PDF
  • Analizzare statisticamente i dati numerici
  • Visualizzare grafici per una migliore comprensione dei dati
  • Inviare report direttamente via email

3. Installazione

Windows

  1. Scaricare il file Reporting-Tool-Setup-1.1.0.exe dalla sezione download
  2. Eseguire il file di installazione facendo doppio clic
  3. Seguire le istruzioni a schermo per completare l'installazione
  4. Avviare l'applicazione dal menu Start o dal desktop

macOS

  1. Scaricare il file Reporting-Tool-1.1.0.dmg dalla sezione download
  2. Aprire il file DMG facendo doppio clic
  3. Trascinare l'icona dell'applicazione nella cartella Applicazioni
  4. Avviare l'applicazione dal Launchpad o dalla cartella Applicazioni
  5. Al primo avvio, potrebbe essere necessario autorizzare l'apertura nelle Preferenze di Sicurezza

Linux

  1. Scaricare il file Reporting-Tool-1.1.0.AppImage dalla sezione download
  2. Rendere il file eseguibile con il comando: 
    chmod +x Reporting-Tool-1.1.0.AppImage
  3. Eseguire l'applicazione con il comando: 
    ./Reporting-Tool-1.1.0.AppImage
  4. Opzionalmente, installare globalmente: 
    sudo mv Reporting-Tool-1.1.0.AppImage /usr/local/bin/reporting-tool
sudo chmod +x /usr/local/bin/reporting-tool

4. Guida all'Uso

4.1. Selezione dei File

Area di selezione dei file

Per importare i file JSON, XML o CSV:

  • Clicca sull'area di trascinamento oppure
  • Trascina direttamente uno o più file nell'area designata
  • Puoi selezionare più file tenendo premuto CTRL o SHIFT durante la selezione
  • I file selezionati verranno mostrati con nome e dimensione
  • Puoi anche accedere ai file recenti dal menu File

Suggerimento: Per una migliore performance, è consigliabile processare file di dimensioni simili contemporaneamente.

4.2. Configurazione dei Campi e Filtri

Area di configurazione dei campi e filtri

Per configurare i campi da estrarre e i filtri:

  1. Seleziona o deseleziona i campi che vuoi estrarre:
    • descriptionMerchant
    • Addebito
    • IBAN
    • fee scadenza
    • totale transazioni
    • descrizione
  2. Attiva l'opzione "Applica filtro ai risultati" se necessario
  3. Configura il filtro selezionando:
    • Campo su cui filtrare
    • Operatore di confronto (>, <, =, >=, <=, contiene)
    • Valore di confronto

Suggerimento: Nella versione 1.1.0 è possibile salvare configurazioni di filtri frequentemente utilizzate come preset [BETA].

4.3. Elaborazione dei File

Processo di elaborazione dei file

Per avviare l'elaborazione dei file:

  1. Clicca sul pulsante "Elabora i file JSON"
  2. Una barra di progresso mostrerà l'avanzamento dell'operazione
  3. Al termine, verrà visualizzata una tabella con i dati estratti
  4. L'applicazione mostrerà il numero totale di record trovati

Nota: L'elaborazione di file di grandi dimensioni potrebbe richiedere più tempo. Non chiudere l'applicazione durante il processo.

4.4. Esportazione dei Risultati

Opzioni di esportazione

Per esportare i dati estratti:

  1. Controlla l'anteprima dei dati nella tabella dei risultati
  2. Scegli il formato di esportazione desiderato:
    • CSV: clicca sul pulsante "Esporta come CSV"
    • XLSX: clicca sul pulsante "Esporta come XLSX"
    • PDF: disponibile dal menu File > "Esporta come PDF"
  3. Seleziona la posizione in cui salvare il file
  4. Attendi il completamento dell'esportazione

Per inviare i risultati via email:

  1. Configura prima il servizio email (vedi sezione 5)
  2. Clicca su "Invia Report via Email" nel menu Strumenti
  3. Compila i campi del form (destinatari, oggetto, messaggio)
  4. Seleziona il formato di allegato desiderato
  5. Clicca su "Invia" per procedere

4.5. Analisi Statistica

Visualizzazione delle statistiche

Per generare statistiche sui dati estratti:

  1. Seleziona il campo numerico per l'analisi dal menu a tendina (es. "Addebito")
  2. Clicca sul pulsante "Genera statistiche"
  3. I risultati statistici verranno visualizzati, inclusi:
    • Conteggio dei valori
    • Valore minimo
    • Valore massimo
    • Media
    • Somma totale
    • Deviazione standard
  4. I grafici mostreranno:
    • Istogramma di distribuzione dei valori
    • Grafico a barre per minimo, massimo e media

Suggerimento: Per un'analisi più approfondita, utilizza la dashboard interattiva accessibile dal menu principale.

5. Configurazione Email

Schermata di configurazione email

Per configurare l'invio di report via email:

  1. Accedi al menu "Strumenti" > "Configura Email"
  2. Seleziona il servizio email desiderato:
    • Gmail
    • Outlook
    • SMTP personalizzato
  3. Inserisci le credenziali di accesso:
    • Indirizzo email
    • Password o password app
  4. Per SMTP personalizzato, inserisci anche:
    • Server SMTP
    • Porta SMTP
  5. Clicca su "Verifica Connessione" per testare la configurazione
  6. Clicca su "Salva" per memorizzare le impostazioni

Nota sulla sicurezza: Le credenziali email vengono salvate localmente sul computer dell'utente. Si consiglia di utilizzare una password specifica per l'applicazione o l'autenticazione a due fattori ove possibile.

6. Risoluzione dei Problemi

L'applicazione non riesce ad aprire i file

Possibili cause:

  • Il file non è in un formato JSON, XML o CSV valido
  • Il file è danneggiato
  • Non hai i permessi necessari per accedere al file

Soluzioni:

  • Verifica che il file sia valido utilizzando un editor JSON/XML/CSV
  • Prova a riaprire il file originale e salvarlo nuovamente
  • Verifica i permessi del file nel sistema operativo
I dati estratti sono incompleti

Possibili cause:

  • Il formato dei dati potrebbe non essere riconosciuto correttamente
  • I campi potrebbero avere nomi o strutture diverse da quelle attese
  • Il file potrebbe contenere formattazioni speciali non supportate

Soluzioni:

  • Verifica che i file contengano un campo "descriptionMerchant" o simile
  • Controlla la struttura dei dati nel file originale
  • Prova a normalizzare il formato del file prima dell'importazione
L'invio email non funziona

Possibili cause:

  • Configurazione del server email errata
  • Credenziali non valide
  • Impostazioni di sicurezza dell'account email restrittive
  • Allegati troppo grandi

Soluzioni:

  • Verifica le impostazioni del server e le credenziali
  • Per Gmail e altri provider, abilita l'accesso per app meno sicure o usa una password app
  • Riduci la dimensione degli allegati esportando solo i campi necessari
  • Controlla la connessione internet
L'applicazione si blocca durante l'elaborazione

Possibili cause:

  • File troppo grandi che superano la memoria disponibile
  • Troppi file elaborati contemporaneamente
  • Sistema con risorse limitate

Soluzioni:

  • Suddividi i file in parti più piccole
  • Elabora meno file alla volta
  • Chiudi altre applicazioni per liberare memoria
  • Aggiorna a un sistema con più RAM

7. Domande Frequenti

Come posso velocizzare l'elaborazione di file di grandi dimensioni?

Per migliorare le prestazioni con file di grandi dimensioni:

  • Chiudi altre applicazioni per liberare memoria
  • Elabora un numero limitato di file alla volta
  • Seleziona solo i campi necessari per l'estrazione
  • Considera di suddividere i file molto grandi in parti più piccole
  • Utilizza un sistema con più RAM (almeno 8 GB consigliati)
L'applicazione è compatibile con tutti i tipi di file JSON?

L'applicazione è progettata per riconoscere la maggior parte dei formati JSON standard, ma con alcune limitazioni:

  • I file JSON devono contenere un campo "descriptionMerchant" o simile per l'estrazione
  • Strutture JSON molto complesse o nidificate potrebbero non essere completamente supportate
  • Formati JSON non standard potrebbero richiedere una pre-elaborazione
  • La versione 1.1.0 ha migliorato il supporto per vari formati JSON
Posso programmare l'elaborazione automatica di file?

Attualmente, l'elaborazione automatica programmata non è disponibile nella versione 1.1.0, ma è pianificata per le versioni future. Nell'attesa, puoi:

  • Utilizzare script del sistema operativo per avviare l'applicazione a orari programmati
  • Salvare i preset di filtri più utilizzati per velocizzare le operazioni manuali
  • Seguire gli aggiornamenti dell'applicazione per le nuove funzionalità
Come posso personalizzare i campi estratti?

Per personalizzare i campi estratti:

  • Utilizza la sezione "Campi da estrarre" nella configurazione
  • Seleziona o deseleziona le caselle di controllo per ogni campo
  • Solo i campi selezionati saranno inclusi nei risultati e nelle esportazioni
  • Per funzionalità di personalizzazione avanzata, consulta la documentazione tecnica
  • Nella versione 1.1.0, i preset di filtri [BETA] consentono di salvare anche configurazioni di campi
I miei dati sono sicuri quando uso la funzionalità di invio email?

La sicurezza dei dati è una priorità:

  • Tutte le credenziali email sono salvate localmente sul tuo computer
  • L'applicazione utilizza connessioni sicure (TLS/SSL) per l'invio delle email
  • Nessun dato viene mai inviato ai nostri server
  • Si consiglia di utilizzare password app o autenticazione a due fattori
  • Per dati particolarmente sensibili, considera l'utilizzo di file criptati
